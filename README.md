# hypr-dark-frost-xdotfiles

Dotfiles for my Hyprland setup on Arch: dark grey theme, frosted windows, and a bunch of shell scripts that keep GTK, Kitty, Rofi, Firefox, VS Code, and Discord in sync.

I use [Ambxst](https://github.com/Axenide/Ambxst) as the shell. Colors and fonts come from there — `~/.cache/ambxst/colors.json` and `~/.config/ambxst/config/theme.json`. This repo is the layer on top that reads those files and writes app configs. Without Ambxst running, the sync scripts won't have much to work with.

## Install

Needs Arch, Hyprland, Ambxst, and `jq`. For fonts I pull SF Pro from the `apple-fonts` AUR package (optional). Fastfetch and Starship come from extra.

```bash
git clone <repo-url> ~/Projects/hypr-dark-frost-xdotfiles
~/Projects/hypr-dark-frost-xdotfiles/install.sh
~/.local/bin/install-shell-hypr-dark-frost-xdotfiles
hyprctl reload
```

Run `install-fonts-hypr-dark-frost-xdotfiles` if you want SF Pro. Restart apps or log out after install.

## What's in here

```
bin/       sync scripts and app launchers
share/     font-stack.sh (shared by the sync scripts)
config/    hypr, kitty, rofi, shell, electron
install.sh copies everything into ~/.local/bin and ~/.config
```

Desktop entries (`code`, `firefox`, `discord`) are generated by `install.sh` into `~/.local/share/applications/`. Paths use `$HOME` at install time — nothing username-specific is committed.

## Scripts

Names are long on purpose so they don't clash with anything else in `PATH`.

`sync-fonts-hypr-dark-frost-xdotfiles` is the one to run manually when the theme changes. It calls the gtk/rofi/kitty/vscode/firefox/shell syncs and updates fontconfig.

Individual syncs exist if you only want to refresh one app: `sync-gtk-hypr-dark-frost-xdotfiles`, `sync-firefox-hypr-dark-frost-xdotfiles`, and so on.

Hyprland autostart runs `sync-fonts-hypr-dark-frost-xdotfiles` and `sync-discord-hypr-dark-frost-xdotfiles` at login.

`code`, `firefox`, and `discord` in `~/.local/bin` are symlinks to the wrapper scripts that sync then launch.

## Fonts

Defaults to SF Pro Text and SF Mono at 14px, read from Ambxst's `theme.json`. If those aren't installed, the font stack script falls back to whatever is available (Inter, JetBrains Mono, etc.).
